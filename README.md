# TalleresAquitectura

Pequeño sitio estático para la práctica de Talleres de Arquitectura.

Cómo ejecutar en Windows (PowerShell):

1. Abre PowerShell y ve al directorio del proyecto:

	cd 'c:\Users\kenek\OneDrive\Documentos\GitHub\EcoViajes\EcoViajes\EcoViajes\TalleresAquitectura'

2. Arranca un servidor HTTP simple (requiere Python 3):

	python -m http.server 8000

	Esto sirve los archivos en http://localhost:8000

3. Abrir la página de login en el navegador:

	

Notas:
- Para detener el servidor, presiona Ctrl+C en la terminal donde lo iniciaste.
- Si no tienes Python, puedes usar cualquier servidor estático (extensiones de VS Code, `http-server` de npm, etc.).

Si quieres, puedo añadir un archivo de comandos o una tarea de VS Code para arrancar el servidor automáticamente.
